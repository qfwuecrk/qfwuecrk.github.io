# 数据结构

数据结构在计算机科学中是一门综合性的专业基础课。可以认为数据结构是介于数学、计算机硬件和软件三者之间的一门核心课程。

``` mermaid
    数据结构与算法 --> 算法分析与设计
    数据结构与算法 --> 计算复杂性理论
    数据结构与算法 --> 数据库（线性表、多链表、排序、B+数）
    数据结构与算法 --> 操作系统（队列、存储管理表、排序、目录树）
    数据结构与算法 --> 编译原理（字符串、栈、哈希表、语法树）
```
