# 图

图的应用极为广泛，已渗入到诸如物理、化学、电信工程、计算机科学，以及数学等其他分支中。在**离散数学**中，图论是专门研究图的性质的数学分支，而在**数据结构**中，则应用图论的知识讨论如何在计算机上实现图的操作，因此主要学习图的存储结构，以及若干图的操作的实现。
